1단계 목표: 기본 Monster 클래스 만들기

이번 단계에서는 우리 게임에 등장할 모든 몬스터의 가장 기본적인 설계도를 만드는 작업을 할 겁니다. 
이 설계도를 클래스(Class)라고 부르죠. 이 클래스에는 몬스터가 가져야 할 기본적인 정보(속성)와 기능(메서드)을 정의할 거예요.

당신이 해야 할 일은 다음과 같습니다:

Monster.java 파일을 생성하세요.

Monster 클래스 안에 다음 속성(인스턴스 변수)을 선언하세요.

name (이름): String 타입

hp (체력): int 타입

maxHp (최대 체력): int 타입 (나중에 체력을 회복할 때 필요해요!)

attack (공격력): int 타입

defense (방어력): int 타입

생성자(Constructor)를 만드세요.

이 생성자는 name, hp, attack, defense 값을 파라미터로 받아서 클래스의 속성들을 초기화해야 합니다.

maxHp는 생성 시의 hp와 동일한 값으로 초기화해주세요.

몬스터의 정보를 보여주는 info() 메서드를 만드세요.

이 메서드는 현재 몬스터의 이름, 체력(현재/최대), 공격력, 방어력을 보기 좋게 콘솔에 출력하는 기능을 합니다.
