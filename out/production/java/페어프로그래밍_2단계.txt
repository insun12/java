2단계 목표: 몬스터 전투 로직 구현하기

이제 우리가 만든 Monster 객체 두 개를 생성해서 서로 싸움을 붙여보겠습니다. 
이 과정에서 연산자와 제어문(while, if)을 사용하게 될 겁니다.

당신이 해야 할 일은 다음과 같습니다:

GameManager.java 파일을 새로 만드세요. (실행을 위한 main 메서드를 포함해야 합니다.)

main 메서드 안에서 Monster 객체 두 개를 생성하세요.
예: monster1 = 슬라임 (체력 30, 공격력 8, 방어력 3)
예: monster2 = 오크 (체력 40, 공격력 12, 방어력 5)

한 몬스터의 hp가 0 이하가 될 때까지 전투를 계속하는 while 반복문을 만드세요.
	반복문 안에서 두 몬스터가 서로 한 번씩 공격하도록 코드를 작성하세요.
	monster1이 monster2를 공격하고, 그 결과를 출력합니다.
	monster2가 monster1을 공격하고, 그 결과를 출력합니다.
	데미지 계산 공식: 피해량 = 공격자의 attack - 방어자의 defense (만약 피해량이 0보다 작으면 0으로 처리해 주세요.)
	피해량만큼 상대방의 hp를 setHp() 메서드를 이용해 깎아주세요.

반복문이 끝나면, 어떤 몬스터가 승리했는지 알려주는 메시지를 출력하세요.

<추가>
전투가 너무 순식간에 끝나버리면 재미가 없죠. 
공격을 주고받는 턴 사이에 잠시 멈춤(딜레이)을 주면 훨씬 더 박진감 넘치는 전투 과정을 볼 수 있습니다.

Thread.sleep() 메서드를 사용하면 되고, try-catch 문으로 감싸주어야 합니다.
sleep 기능을 구현하는 코드는 다음과 같습니다. 
try {
    // 1000 밀리초 = 1초 동안 실행을 멈춥니다.
    Thread.sleep(1000); 
} catch (InterruptedException e) {
    // sleep 도중 방해를 받았을 때 처리할 코드를 여기에 작성할 수 있습니다.
    // 지금은 비워두어도 괜찮습니다.
    e.printStackTrace();
}
